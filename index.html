Steam Bookmarks

Steam Bookmarks is simple a site with a modern flat design that lets you create an account and save bookmarks.

links

Live Demo View Code

Story

This is one of my very first projects made around 2016-2017

Steam in-game browser is a very basic browser that you can use while playing games without having to minimize them, but it doesn't have a bookmark system, which is very inconvenient. imagine having to manually type something like "www.reddit.com/r/GlobalOffensive" just to see your favorite game's latest news or guides and every time you enter the game over and over again.

And so comes the purpose of this site, set it as your steam browser's homepage, create an account, and add your bookmarks.

What i used

Php MySql Javascript jQuery AJAX HTML CSS

I used jquery (3.3.1) AJAX functions to add/delete/re-arrange bookmarks through the backend api built with php.

The data is saved in a Mysql database, see image below

Security checklist

  • Anti SQL injection
  • Anti XSS attacks
  • Passwords hashed in the database
  • Backend form validation
